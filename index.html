Key Features

Journal Entries

Create, edit, publish, and archive journal entries with rich text editor and automatic versioning.

Mood Analytics

Track emotions with interactive dashboards including pivot tables, graphs, and trend analysis.

Full-Text Search

Advanced search across titles, content, tags, and notebooks using PostgreSQL vectors.

Version Control

Automatic version history with diff comparison, restore points, and changelog.

Export Options

Export entries as PDF or Markdown with custom formatting and batch operations.

Tag Management

Categorize entries with colored tags, smart filters, and tag analytics.

Installation

Requirements

  • Odoo 15.0 or later
  • PostgreSQL 12 or later
  • Python 3.8+

Quick Installation

# 1. Clone the repository
git clone https://github.com/AyeshaChughtaiIsMe/odoo-journal.git

# 2. Copy to Odoo addons directory
cp -r odoo-journal/journal /path/to/odoo/addons/

# 3. Update Odoo modules
./odoo-bin -u base

# 4. Install Journal module
./odoo-bin -i journal

Configuration

# In Odoo, after installation:
1. Go to Apps → Journal
2. Create your first notebook
3. Start writing entries
4. Configure search preferences
5. Set up automated backups
